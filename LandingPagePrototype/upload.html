NEX Sandbox

"NEX sandbox is a prototyping environment in which users have access to all NEX data, tools and smaller scale computing. Prototypes from the NEX sandbox can be easily scaled up and deployed on the NASA High Performance Computing (HPC) system. Any NASA researcher can request access to the sandbox, and if approved should have access within 24 hours. Any non-NASA user in the United States can request access to the sandbox, and if approved should have access within 48 hours. To find out more about the NEX sandbox and how to request access, visit the NEX portal and/or the sandbox documentation ".
