    Node Help Manual

Each box in this section corresponds to a column in your specified spreadsheet. Each column should correspond to a set of nodes, or nodeset. If you check the "is a nodeset?" box, then every value in that column will represent a node in the network. If you do not, then the values in that column will not be in the network. You may also elect to rename nodesets; the default value is the column name in the spreadsheet. The next page will allow you to select which relationships between the nodesets you would like to visualize and analyze.
